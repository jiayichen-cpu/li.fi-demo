# 📖 如何预览落地页 - 详细步骤

## 方法一：使用终端/命令行（推荐）

### 什么是终端/命令行？
终端是一个文本界面，可以输入命令来操作电脑。在 Mac 上：
- 按 `Command + 空格键` 打开 Spotlight
- 输入 "终端" 或 "Terminal"
- 点击打开终端应用

### 详细步骤：

#### 第 1 步：打开终端
- Mac: 按 `Command + 空格`，输入 "Terminal"，回车
- 或者：应用程序 → 实用工具 → 终端

#### 第 2 步：进入项目文件夹
在终端中输入以下命令（一行一行输入，每行后按回车）：

```bash
cd /Users/yael/Downloads/lifi-carnival-landing
```

**解释：** `cd` 是"进入文件夹"的意思，后面是文件夹的路径

#### 第 3 步：安装依赖（只需要做一次）
```bash
npm install
```

**解释：** 这会下载项目需要的所有代码库。需要等待几分钟。

**看到什么说明成功了？**
- 看到很多行文字滚动
- 最后没有错误信息
- 出现类似 `added 234 packages` 的提示

#### 第 4 步：启动预览服务器
```bash
npm run dev
```

**解释：** 这会启动一个本地服务器，让你可以在浏览器中查看页面

**看到什么说明成功了？**
```
  ▲ Next.js 16.0.7
  - Local:        http://localhost:3000
  - Network:      http://192.168.x.x:3000
```

#### 第 5 步：打开浏览器
1. 打开浏览器（Safari、Chrome、Firefox 都可以）
2. 在地址栏输入：`http://localhost:3000`
3. 按回车

**完成！** 现在你应该能看到落地页了。

## 方法二：使用 VS Code（如果有的话）

如果你安装了 Visual Studio Code：

1. 打开 VS Code
2. 文件 → 打开文件夹 → 选择 `/Users/yael/Downloads/lifi-carnival-landing`
3. 点击顶部菜单：终端 → 新建终端
4. 在底部终端窗口中输入：
   ```bash
   npm install
   ```
   等待完成
5. 然后输入：
   ```bash
   npm run dev
   ```
6. 点击终端中显示的 `http://localhost:3000` 链接，或手动在浏览器打开

## 方法三：检查是否已经安装过依赖

如果你不确定是否已经安装过依赖，可以：

1. 打开终端
2. 输入：
   ```bash
   cd /Users/yael/Downloads/lifi-carnival-landing
   ls node_modules
   ```
3. 如果看到很多文件夹列表，说明已经安装过了，可以直接运行 `npm run dev`
4. 如果看到 "No such file or directory"，需要先运行 `npm install`

## 常见问题

### Q: 输入命令后没反应？
A: 确保在正确的文件夹中，先运行 `cd /Users/yael/Downloads/lifi-carnival-landing`

### Q: 显示 "command not found: npm"？
A: 需要先安装 Node.js：
   - 访问 https://nodejs.org
   - 下载并安装 LTS 版本
   - 安装后重启终端

### Q: 端口 3000 被占用？
A: Next.js 会自动使用下一个可用端口（3001, 3002 等），注意终端中显示的端口号

### Q: 如何停止服务器？
A: 在终端中按 `Ctrl + C`（Mac 上是 `Control + C`）
