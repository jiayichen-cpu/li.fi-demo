# 解决 npm install 卡住的问题

## 方法一：使用国内镜像源（推荐，如果在中国）

如果 npm install 一直卡住，可能是网络问题。可以尝试使用国内镜像源：

### 步骤：

1. **停止当前的安装**（如果还在运行）：
   - 在终端按 `Ctrl + C`

2. **切换到淘宝镜像源**：
   ```bash
   npm config set registry https://registry.npmmirror.com
   ```

3. **重新安装**：
   ```bash
   cd /Users/yael/Downloads/lifi-carnival-landing
   npm install
   ```

4. **如果成功了，以后可以继续用这个镜像源**

### 如果想恢复原镜像源：
```bash
npm config set registry https://registry.npmjs.org/
```

---

## 方法二：使用 yarn（替代方案）

如果 npm 一直有问题，可以尝试使用 yarn：

1. **安装 yarn**：
   ```bash
   npm install -g yarn
   ```

2. **使用 yarn 安装依赖**：
   ```bash
   cd /Users/yael/Downloads/lifi-carnival-landing
   yarn install
   ```

3. **启动预览**：
   ```bash
   yarn dev
   ```

---

## 方法三：清理缓存后重试

1. **清理 npm 缓存**：
   ```bash
   npm cache clean --force
   ```

2. **删除 node_modules 文件夹**（如果存在）：
   ```bash
   cd /Users/yael/Downloads/lifi-carnival-landing
   rm -rf node_modules
   ```

3. **重新安装**：
   ```bash
   npm install
   ```

---

## 方法四：检查网络连接

确保你的网络连接正常：

```bash
# 测试网络
ping registry.npmjs.org
```

如果 ping 不通，可能是网络问题。

---

## 方法五：使用代理（如果有）

如果你有代理，可以设置：

```bash
npm config set proxy http://your-proxy:port
npm config set https-proxy http://your-proxy:port
```

---

## 最快解决方案（推荐）

**直接使用淘宝镜像源**：

```bash
# 1. 停止当前安装（Ctrl + C）

# 2. 设置镜像源
npm config set registry https://registry.npmmirror.com

# 3. 重新安装
cd /Users/yael/Downloads/lifi-carnival-landing
npm install
```

这通常能解决大部分网络问题。
