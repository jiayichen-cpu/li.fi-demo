## To Run Locally

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Open browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Project Structure

- **app/layout.tsx** - Basic root layout with LanguageProvider
- **app/page.tsx** - Landing page with placeholder sections
- **components/** - All placeholder components (Hero, DailyRewards, TeamLeaderboard, HowToParticipate, FAQ)
- **lib/dictionary.ts** - Minimal translation dictionary (en/zh)
- **lib/i18n.tsx** - i18n context provider
- **components/ui/** - shadcn/ui components (Card, Accordion, Button)
